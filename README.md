# WinForms-GMap-Integration

C# ile geliştirilmiş, GMap.NET kütüphanesi kullanılarak hazırlanmış basit ve interaktif bir harita masaüstü uygulamasıdır.

##  Kullanılan Teknolojiler

- C#
- .NET Framework (WinForms)
- GMap.NET

##  Temel Özellikler

- Harita üzerinde konum görüntüleme
- Enlem ve boylam bilgisi girerek haritada konum bulma
- Girilen koordinata marker (işaretçi) ekleme
- Harita üzerinde yakınlaştırma ve sürükleme desteği

##  Kurulum

```bash
1. Bu repoyu klonlayın:
   git clone https://github.com/github_irmakalty/WinForms-GMap-Integration.git

2. Visual Studio ile .sln dosyasını açın

3. Gerekli NuGet paketlerini yükleyin:
   - GMap.NET.WindowsForms
   - GMap.NET.Core

4. Projeyi çalıştırın (F5)
